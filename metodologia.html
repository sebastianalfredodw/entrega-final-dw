Estrategia

En este primer paso, es donde se toman las decisiones del proyecto en conjunto con el cliente, luego se procede a la planificación posteriormente la ejecución y luego a generar los avances que permitan la finalizacion del proceso.

Esta fase trata sobre la maquetación; definir su estructura web, colores, animaciones, funciones, etc. el conjunto de estos atributos y elementos conforman el diseño el cual se preparan para la siguiente fase.

03

Desarrollo

Luego de haber generado la Esrategia y definido el Diseño, se genera la estructuración del sitio web en base a codigo de maquetación HTML, definir estilos con CSS y aplicar animaciones via Javascript. Para lograr animaciones más prolijas, se deben aplicar librerias y frameworks mas complejos como ReactJS.

Finalmente, concluimos entregando el proyecto. Al cliente se le recomienda la supervisión o mantención periodica del servicio. El lenguaje, los plugins y los mismos servidores requieren revisión constante para mantener el sitio en optimas condiciones.
